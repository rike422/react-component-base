# [tanmono](http://rike422.github.io/tanmono/)

[demo](https://rike422.github.io/tanmono/)

A project template for creating stand-alone my React components. It is pre-configured with

- React
- Babel
- CSSModules
- Webpack
- karma
- mocha + power-assert

## Commands

### Generator

```
npm run generate <moduleName>
```
